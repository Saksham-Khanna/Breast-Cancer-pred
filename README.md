# Breast Cancer Prediction App

A **Streamlit-based web app** that predicts whether a tumor is **Positive (Cancerous)** or **Negative (Non-Cancerous)** based on key features extracted from breast cancer datasets.

---

## Features

- Uses **5 key features** of tumors:  
  - `radius_mean`  
  - `texture_mean`  
  - `perimeter_mean`  
  - `area_mean`  
  - `smoothness_mean`  
- **Interactive sliders** in the sidebar to adjust feature values.  
- **Predict button** to classify tumor as Positive or Negative.  
- **Dynamic background color** changes based on prediction:  
  - Light red for Positive (Cancerous)  
  - Light green for Negative (Non-Cancerous)  
- Displays a **representative image** (`pos.jpeg` / `neg.jpeg`) based on prediction.  
- Simple and professional UI.
